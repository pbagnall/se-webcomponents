Using the date picker

You can make the component available simply by adding this script into your head element.

<script type='module' src='/components/datepicker.js'></script>

And then simply use by insert the se-datepicker tag into your HTML like this

<se-datepicker date='2023-5-12'></se-datepicker>

Of course, that's not very helpful, you need to be able to get the value from the datepicker. You can do that exactly the same way you would with a native date picker.

  • In a form, set the name and the value will be passed with the form data.
  • Read the element's value property
  • Listen to the onchange events
<se-datepicker date='2023-5-12' name='appointment'></se-datepicker>

Styling the date picker

--oddPeriodBackground
background colour for odd months and years
--evenPeriodBackground
background colour for even months and years
--toolbarBackground
background for the toolbar with the days of the week and up/down month/year buttons
--foreground
Colour of dates, days of week, and all other text
--hoverColor
Text colour of dates when being hovered over
--hoverBackgroundColor
Background colour of dates when being hovered over
--selectedColor
Text colour of selected date
--selectedBackgroundColor
Background colour of selected date
--todayBorderColor
Colour of border around today's date
